# 示例分析:
## 控制流图
```
块0: [GetParameter(r1, 0), GetParameter(r2, 1)]
块1: [Add(r3, r1, r2), Mv(r4, r3)]
块2: [Return(r4)]

```

# 分析结果
## 块0分析
```rust
// 指令1: GetParameter(r1, 0)
killed.insert(r1);           // r1被定义
defined_in[r1].insert(0);    // r1在块0定义

// 指令2: GetParameter(r2, 1)  
killed.insert(r2);           // r2被定义
defined_in[r2].insert(0);    // r2在块0定义

```

## 块1分析
```rust
// 指令1: Add(r3, r1, r2)
test_and_insert!(r1);        // r1在killed中？否 → exposed.insert(r1)
test_and_insert!(r2);        // r2在killed中？否 → exposed.insert(r2)  
killed.insert(r3);           // r3被定义
defined_in[r3].insert(1);    // r3在块1定义

// 指令2: Mv(r4, r3)
test_and_insert!(r3);        // r3在killed中？是 → 不插入exposed
killed.insert(r4);           // r4被定义
defined_in[r4].insert(1);    // r4在块1定义

```

## 块2分析
```rust
// 指令1: Return(r4)
test_and_insert!(r4);        // r4在killed中？否 → exposed.insert(r4)
```


## 分析结果:
```
exposed = {r1, r2, r4}  // 全局寄存器列表
defined_in = {
    r1: [0],  // r1在块0定义
    r2: [0],  // r2在块0定义  
    r3: [1],  // r3在块1定义
    r4: [1]   // r4在块1定义
}
```
