# 例子
## 输入（线性IR）
```rust
li rd1, #0
j @L1
@L1:
beq rd1, rd2, @L2, @L3  
@L2:
return rd1
@L3:
add rd1, rd1, #1
j @L1
```

## 构建的控制流图
```rust
块0: [li, j] → 块1
块1 (L1): [beq] → 块2, 块3  
块2 (L2): [return] → EXIT
块3 (L3): [add, j] → 块1
EXIT: []
```

# 对后续代码的关键作用
## 1. SSA转换的基础
__支配关系计算__：基于控制流图计算支配者和支配边界

__Phi函数插入__：在支配边界处插入Phi函数实现SSA形式

__变量重命名__：基于控制流进行SSA变量重命名


## 2. 数据流分析的前提
__活跃变量分析__：确定变量在基本块出口处的活跃状态

__寄存器分配__：为图着色寄存器分配提供基础

__死代码消除__：识别和删除不可达代码

## 3. 优化算法的输入
__常量传播__：基于控制流传播常量值

__公共子表达式消除__：识别跨基本块的公共表达式

__循环优化__：识别循环结构进行优化

## 4. 代码生成的准备
__基本块调度__：优化基本块的执行顺序

__跳转优化__：优化条件分支和无条件跳转

__指令选择__：基于控制流选择最优指令序列
