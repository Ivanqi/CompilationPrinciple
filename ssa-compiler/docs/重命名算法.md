# 控制流图
```
块0 (ENTRY) → 块1
块1 → 块2, 块3
块2 → 块4
块3 → 块4
块4 → EXIT
```
初始状态:
- 全局变量：`g`
- 初始Phi函数：`块4: g4 = Φ(g, g, g)`

# 重命名过程
## 阶段1：处理块0（入口块）
```rust
// 为全局变量g压栈
names[g] = [g]  // 初始版本

// 处理指令：假设块0中有 GetParameter(g, 0)
set_name!(g);  // g → g0
// names[g] = [g0]
```

## 阶段2：处理块1
```rust
// 为全局变量g压栈
names[g] = [g0, g0]  // 压栈

// 处理指令：假设块1中有 Add(g, x, y)
update_name!(x);  // 使用当前版本
update_name!(y);  // 使用当前版本  
set_name!(g);     // g0 → g1
// names[g] = [g0, g1]

// 递归处理被支配的块（块2、块3、块4）
```

## 阶段3：处理块2
```rust
// 为全局变量g压栈
names[g] = [g0, g1, g1]  // 压栈

// 处理指令：假设块2中有 Mv(g, 5)
set_name!(g);  // g1 → g2
// names[g] = [g0, g1, g2]

// 更新块4中Phi函数的参数
// 块4的Phi函数：g4 = Φ(g, g, g)
// 当前块是块2，在块4的前驱列表中的位置是0
// 更新args[0] = g2
```

## 阶段4：处理块3
```rust
// 为全局变量g压栈
names[g] = [g0, g1, g1]  // 压栈（从块1的栈状态开始）

// 处理指令：假设块3中有 Mv(g, 10)
set_name!(g);  // g1 → g3
// names[g] = [g0, g1, g3]

// 更新块4中Phi函数的参数
// 块4的Phi函数：g4 = Φ(g2, g, g)
// 当前块是块3，在块4的前驱列表中的位置是1
// 更新args[1] = g3
```

## 阶段5：处理块4
```rust
// 为全局变量g压栈
names[g] = [g0, g1, g1]  // 压栈（从块1的栈状态开始）

// 处理Phi函数：g4 = Φ(g2, g3, g)
// 重命名Phi函数的目标寄存器
set_name!(g);  // g1 → g4
// names[g] = [g0, g1, g4]

// 更新Phi函数的最后一个参数（来自块1）
// 当前块是块4，在块4的前驱列表中的位置是2
// 更新args[2] = g1
```

## 最终结果
```
块0: g0 = GetParameter(g, 0)
块1: g1 = Add(g0, x, y)
块2: g2 = Mv(g1, 5)
块3: g3 = Mv(g1, 10)  
块4: g4 = Φ(g2, g3, g1)
```
