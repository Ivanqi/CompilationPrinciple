# 迭代计算过程
## 迭代循环
```rust
while changed {
    changed = false;
    for (i, block) in self.blocks.iter().enumerate() {
        let mut new: HashSet<_> = if let Some(fst) = selector(block).first() {
            selector(block)
                .iter()
                .map(|n| &doms[*n])
                .fold(doms[*fst].clone(), |acc, next| {
                    acc.intersection(next).cloned().collect()
                })
        } else {
            HashSet::new()
        };
        new.insert(i);
        if new != doms[i] {
            changed = true;
            doms[i] = new;
        }
    }
}

```

## 迭代过程
### 第0次迭代(初始化)
```
doms[0] = {0}          // 入口只支配自己
doms[1] = {0,1,2,3,4}  // 初始支配所有块
doms[2] = {0,1,2,3,4}
doms[3] = {0,1,2,3,4}  
doms[4] = {0,1,2,3,4}
```

### 第1次迭代
#### 计算块1:
```
前驱: {0}
交集: doms[0] = {0}
加上自己: {0, 1}
更新：doms[1] = {0,1}
```
#### 计算块2
```
前驱：{1}
交集：doms[1] = {0,1}
加上自己：{0,1,2}
更新：doms[2] = {0,1,2}
```

#### 计算块3
```
前驱：{1}
交集：doms[1] = {0,1}
加上自己：{0,1,3}
更新：doms[3] = {0,1,3}
```
#### 计算块4
```
前驱：{2,3}
交集：doms[2] ∩ doms[3] = {0,1,2} ∩ {0,1,3} = {0,1}
加上自己：{0,1,4}
更新：doms[4] = {0,1,4}

```

### 第2次迭代
所有集合不再变化，算法收敛

## 直接支配者计算
```rust
let apsp_reverse = self.apsp_reverse(postdominance);

for (i, mut set) in doms.into_iter().enumerate() {
    set.remove(&i);
    let idom = set.into_iter().min_by_key(|&v| apsp_reverse[i][v]);
    idoms[i] = idom;
}

```
- `set.remove(&i)`: 移除自己，只保留严格支配者
- `apsp_reverse`: 反向最短路径距离矩阵
- 选择距离最近的支配者作为直接支配者
