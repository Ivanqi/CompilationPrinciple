```rust
// 基本块0: x = 5, y = 10
// 基本块1: if (cond) goto 2 else goto 3
// 基本块2: z = x + y; goto 4
// 基本块3: w = x + y; goto 4
// 基本块4: result = phi(z, w)

```

```rust
// 基本块0: x = 5, y = 10

```

# 算法原理总结
## 识别阶段
- 遍历控制流合并点
- 通过phi翻译识别可用表达式
- 确定可以安全提升的表达式

## 插入阶段
- 在前驱块插入冗余计算
- 在当前块插入phi函数
- 维护SSA形式的正确性

## 传播阶段
- 将新表达式信息传播到支配子节点
- 迭代处理直到收敛

## 优化效果
- 消除部分冗余计算
- 提升循环不变式
- 提高代码执行效率
