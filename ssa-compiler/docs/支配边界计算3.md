# 定义
支配边界(Dominance Frontier): 块 B 的支配边界是所有满足以下条件的块 X 的集合
- B 支配 X 的某个前缀
- B 不严格支配 X

# 示例分析
## 控制流图
```
块0 (ENTRY) → 块1
块1 → 块2, 块3
块2 → 块4
块3 → 块4
块4 → EXIT

```

## 支配树
```
块0 (根节点)
  └── 块1
      ├── 块2
      ├── 块3
      └── 块4
```

## 支配关系
```
idoms[0] = None     // 入口没有支配者
idoms[1] = Some(0)  // 块1的直接支配者是块0
idoms[2] = Some(1)  // 块2的直接支配者是块1
idoms[3] = Some(1)  // 块3的直接支配者是块1
idoms[4] = Some(1)  // 块4的直接支配者是块1

```

# 计算过程分析
## 计算块4的支配边界
```rust
// i = 4 (当前块)
// block.preds = [2, 3] (块4的前驱)

// __外层循环__：遍历所有块作为目标块
// 处理前驱块2：
for &pred in &[2, 3] {
    if let Some(idom) = 4.idom {  // idom = 1
        let mut current = 2;      // 从前驱块2开始
        // __内层循环__：遍历目标块的所有前驱
        // __支配链遍历__：从前驱块开始，沿着支配链向上，直到遇到目标块的直接支配者
        while 1 != 2 {            // 1 != 2，进入循环
            frontier[2].insert(4); // 将块4添加到块2的支配边界
            if let Some(next) = self.blocks[2].idom {  // next = 1
                current = 1;      // 移动到块1
            }
        }
        // 当 current = 1 时，1 == 1，循环结束
    }
}

// 处理前驱块3：
// 同样的逻辑，将块4添加到块3的支配边界

```

## 最终支配边界过:
```
frontier[0] = {}      // 入口块没有支配边界
frontier[1] = {4}     // 块1的支配边界是块4
frontier[2] = {4}     // 块2的支配边界是块4  
frontier[3] = {4}     // 块3的支配边界是块4
frontier[4] = {}      // 块4没有支配边界
```

# 在 SSA 转换中的应用
```rust
// 在SSA转换中
let frontiers = self.get_dominance_frontiers();

for &global in globals.iter() {
    let mut queue: Vec<_> = defined_in[&global].clone();
    while let Some(next) = queue.pop() {
        for &succ in &frontiers[next] {
            // 在支配边界处插入Phi函数
            let entry = SSAOperator::Phi(global, vec![global; self.blocks[succ].preds.len()]);
            if !phis[succ].contains(&entry) {
                phis[succ].push(entry);
                queue.push(succ);
            }
        }
    }
}

```
